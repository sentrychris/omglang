OMG playground

Type any OMG program in the left pane and hit Run. Everything happens in your browser: a JavaScript bundle of the OMG-in-OMG compiler and VM (built by the same self-hosted toolchain that produces the native binaries) reads your source, compiles it to bytecode, and runs that bytecode — same answer the Rust runtime would give, byte-for-byte. The dropdown picks a starter program; edit freely from there.
