Served by OMG

This page, the stylesheet, and the SVG logo above are all served by an HTTP/1.0 server written in OMG. Each request runs in its own forked process, so the server can handle many connections at once even with a single accept loop.

Try it:

  • / — this page
  • /about.html — a second static page
  • /style.css — the stylesheet, served as text/css
  • /logo.svg — the SVG logo, served as image/svg+xml
  • /slow — busy-loops for ~half a second; open it in one tab and reload /about.html in another to see them complete independently.
  • /no-such-page — a 404
