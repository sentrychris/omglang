About

The server lives in tools/web/http_server.omg. It uses six TCP builtins (tcp_listen, tcp_accept, tcp_connect, tcp_read, tcp_write, tcp_close) plus a POSIX fork() for concurrency.

Everything works in both the Rust runtime (omg tools/web/http_server.omg) and as a native-compiled ELF binary (bootstrap/bin/omg --build tools/http_server.omg out).
