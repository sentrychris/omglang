The traditional first post. In OMG it looks like:

```omg
;;;omg

emit "Hello, world!"
```

A few small features used by this very page:

- `**bold**`, `*italic*`, `` `code` `` — all work inline.
- Fenced code blocks with `omg`, `sh`, etc.
- Markdown lists like the one you're reading now.

That's about as deep as the markdown subset goes. See
[md2html.omg](https://github.com/sentrychris/omglang/blob/main/tools/md2html.omg)
for the full list of supported constructs.
