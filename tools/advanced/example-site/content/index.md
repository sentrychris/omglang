This site was generated by [ssg.omg](https://github.com/sentrychris/omglang/blob/main/tools/advanced/ssg.omg),
a small static site generator **written in OMG itself**.

The generator walks `content/`, parses front-matter, converts each
markdown file with the `md2html.omg` library, and applies the
templates in `templates/`. The result lives in `out/`.
